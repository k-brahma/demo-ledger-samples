# 06 在庫評価・滞留チェックツール

## 概要

在庫金額と最終出庫日から滞留在庫や評価減候補を洗い出すツール。

## 起動

```powershell
cd 06_inventory_valuation_checker
python gui.py
```

## 使い方

1. 起動すると `data/inventory.csv` のサンプルデータを自動読み込みします
2. 「CSVを開く」で別のCSVに切り替えられます
3. 左側の一覧で明細を確認し、右側のグラフで傾向を把握します
4. 「結果を保存」で `results/inventory_valuation_report.csv` に出力します

## サンプルデータ列

| 列名 | 内容 |
|------|------|
| 商品コード | 在庫識別子 |
| 商品名 | 商品名 |
| カテゴリ | 商品カテゴリ |
| 数量 | 在庫数量 |
| 単価 | 単価 |
| 最終出庫日 | 最後に出庫した日 |

## 特徴

- 滞留在庫の金額インパクトをすぐ把握できる
- カテゴリ別の在庫偏りも見やすい
- 棚卸や評価減検討のきっかけになる
