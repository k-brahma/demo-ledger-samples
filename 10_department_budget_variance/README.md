# 10 部門別 予実差異分析ツール

## 概要

部門別・科目別の予算と実績を比較し、差異額と差異率の大きい項目を抽出するツール。

## 起動

```powershell
cd 10_department_budget_variance
python gui.py
```

## 使い方

1. 起動すると `data/budget_variance.csv` のサンプルデータを自動読み込みします
2. 「CSVを開く」で別のCSVに切り替えられます
3. 左側の一覧で明細を確認し、右側のグラフで傾向を把握します
4. 「結果を保存」で `results/budget_variance_report.csv` に出力します

## サンプルデータ列

| 列名 | 内容 |
|------|------|
| 部門 | 部門名 |
| 科目 | 費目や収益科目 |
| 予算 | 予算額 |
| 実績 | 実績額 |
| 前年差 | 前年同月との差額 |

## 特徴

- 予実差異と前年差異を同時に見られる
- 大きく振れた費目を部門別に把握しやすい
- 月次報告資料の元ネタとして使いやすい
