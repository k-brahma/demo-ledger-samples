# 07 仕訳異常スコアリングツール

## 概要

起票日時や金額パターンから、監査上気になる仕訳をスコアリングして表示するツール。

## 起動

```powershell
cd 07_journal_entry_anomaly_viewer
python gui.py
```

## 使い方

1. 起動すると `data/journal_entries.csv` のサンプルデータを自動読み込みします
2. 「CSVを開く」で別のCSVに切り替えられます
3. 左側の一覧で明細を確認し、右側のグラフで傾向を把握します
4. 「結果を保存」で `results/journal_anomaly_report.csv` に出力します

## サンプルデータ列

| 列名 | 内容 |
|------|------|
| 伝票No | 仕訳伝票番号 |
| 起票日時 | 起票した日時 |
| 起票者 | 担当者 |
| 借方科目 | 借方科目 |
| 貸方科目 | 貸方科目 |
| 金額 | 仕訳金額 |
| 摘要 | 摘要 |

## 特徴

- 休日起票や深夜起票をすぐ拾える
- 摘要キーワードや端数の特徴もスコアに反映できる
- 監査・内部統制レビューの導入例として使いやすい
