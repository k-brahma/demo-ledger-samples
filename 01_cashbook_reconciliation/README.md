# 01 現金出納帳 突合チェックツール

## 概要

現金出納帳の帳簿残高と実査残高を比較し、過不足や差異の大きい締め日を洗い出すツール。

## 起動

```powershell
cd 01_cashbook_reconciliation
python gui.py
```

## 使い方

1. 起動すると `data/cashbook.csv` のサンプルデータを自動読み込みします
2. 「CSVを開く」で別のCSVに切り替えられます
3. 左側の一覧で明細を確認し、右側のグラフで傾向を把握します
4. 「結果を保存」で `results/cashbook_report.csv` に出力します

## サンプルデータ列

| 列名 | 内容 |
|------|------|
| 日付 | 現金締め日 |
| 担当者 | 出納担当者 |
| 摘要 | 主な入出金内容 |
| 入金 | 当日の入金額 |
| 出金 | 当日の出金額 |
| 帳簿残高 | 出納帳上の残高 |
| 実査残高 | 実際に数えた残高 |

## 特徴

- 小口現金の締め差異を日次で可視化できる
- 担当者別に差異の偏りを確認できる
- 差異幅に応じて正常・警告・要調査を自動判定する
