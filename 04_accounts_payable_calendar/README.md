# 04 支払予定カレンダーツール

## 概要

買掛金や未払金の支払予定を日付順に並べ、支払集中日と遅延リスクを確認するツール。

## 起動

```powershell
cd 04_accounts_payable_calendar
python gui.py
```

## 使い方

1. 起動すると `data/payables.csv` のサンプルデータを自動読み込みします
2. 「CSVを開く」で別のCSVに切り替えられます
3. 左側の一覧で明細を確認し、右側のグラフで傾向を把握します
4. 「結果を保存」で `results/payable_calendar_report.csv` に出力します

## サンプルデータ列

| 列名 | 内容 |
|------|------|
| 支払先 | 仕入先や外注先 |
| 請求日 | 請求書日付 |
| 支払予定日 | 支払予定日 |
| 金額 | 支払予定金額 |
| カテゴリ | 費目カテゴリ |
| 支払状況 | 未払 or 支払済 |

## 特徴

- 資金繰りの山がどの日に集中するかを把握できる
- 未払いのまま期限超過した案件を抽出できる
- カテゴリ別の支払予定額も確認しやすい
