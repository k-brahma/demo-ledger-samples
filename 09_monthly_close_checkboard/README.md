# 09 月次決算 チェックボード

## 概要

月次決算タスクの期限と完了日を管理し、締め遅延や着手漏れを可視化するツール。

## 起動

```powershell
cd 09_monthly_close_checkboard
python gui.py
```

## 使い方

1. 起動すると `data/monthly_close.csv` のサンプルデータを自動読み込みします
2. 「CSVを開く」で別のCSVに切り替えられます
3. 左側の一覧で明細を確認し、右側のグラフで傾向を把握します
4. 「結果を保存」で `results/monthly_close_report.csv` に出力します

## サンプルデータ列

| 列名 | 内容 |
|------|------|
| タスク | 決算タスク名 |
| カテゴリ | 売上、固定資産、税務など |
| 担当者 | 担当者 |
| 期限日 | 期限日 |
| 完了日 | 完了日、未完了なら空欄 |
| 優先度 | 高・中・低 |

## 特徴

- 締め作業の遅れを担当者ごとに確認できる
- 決算チェックリストの簡易版として使える
- 月次決算の進捗会議向けの画面を作りやすい
