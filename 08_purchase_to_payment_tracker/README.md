# 08 発注〜支払 進捗トラッカーツール

## 概要

発注、検収、請求、支払の進捗を1行で見える化し、購買統制の抜け漏れを確認するツール。

## 起動

```powershell
cd 08_purchase_to_payment_tracker
python gui.py
```

## 使い方

1. 起動すると `data/purchase_flow.csv` のサンプルデータを自動読み込みします
2. 「CSVを開く」で別のCSVに切り替えられます
3. 左側の一覧で明細を確認し、右側のグラフで傾向を把握します
4. 「結果を保存」で `results/purchase_flow_report.csv` に出力します

## サンプルデータ列

| 列名 | 内容 |
|------|------|
| 発注No | 案件番号 |
| 仕入先 | 取引先 |
| 発注日 | 発注日 |
| 検収日 | 検収日、未実施なら空欄 |
| 請求日 | 請求書受領日、未着なら空欄 |
| 支払日 | 支払日、未実施なら空欄 |
| 金額 | 案件金額 |

## 特徴

- 未検収請求や未払案件を段階別に見られる
- 案件ごとの現在位置を一覧で把握しやすい
- 購買統制や業務フロー説明のサンプルとして使いやすい
