# 02 銀行明細 消込チェックツール

## 概要

銀行明細と元帳の入出金を突合し、未消込や金額不一致を一覧表示するツール。

## 起動

```powershell
cd 02_bank_statement_matcher
python gui.py
```

## 使い方

1. 起動すると `data/bank_match.csv` のサンプルデータを自動読み込みします
2. 「CSVを開く」で別のCSVに切り替えられます
3. 左側の一覧で明細を確認し、右側のグラフで傾向を把握します
4. 「結果を保存」で `results/bank_match_report.csv` に出力します

## サンプルデータ列

| 列名 | 内容 |
|------|------|
| 日付 | 入出金日 |
| 伝票No | 元帳側の伝票番号 |
| 摘要 | 取引内容 |
| 元帳金額 | 元帳に計上した金額 |
| 銀行金額 | 銀行明細側の金額 |

## 特徴

- 消込漏れや二重計上の候補をすぐ確認できる
- 差額が小さい明細は要確認として切り分けられる
- 銀行残高との差分をCSVで残せる
