# 05 固定資産 減価償却台帳ツール

## 概要

固定資産台帳から定額法ベースの減価償却費、累計償却額、期末簿価を計算するツール。

## 起動

```powershell
cd 05_fixed_asset_depreciation
python gui.py
```

## 使い方

1. 起動すると `data/assets.csv` のサンプルデータを自動読み込みします
2. 「CSVを開く」で別のCSVに切り替えられます
3. 左側の一覧で明細を確認し、右側のグラフで傾向を把握します
4. 「結果を保存」で `results/asset_depreciation_report.csv` に出力します

## サンプルデータ列

| 列名 | 内容 |
|------|------|
| 資産名 | 固定資産の名称 |
| カテゴリ | PC・設備・車両など |
| 取得日 | 資産の取得日 |
| 取得価額 | 取得原価 |
| 残存価額 | 残存価額 |
| 耐用年数 | 年数 |

## 特徴

- 減価償却の基礎計算をGUIで見せやすい
- 除却検討対象や耐用年数終了資産を抽出できる
- 簿価の大きい資産を一覧で確認できる
