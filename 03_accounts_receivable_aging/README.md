# 03 売掛金 滞留年齢表ツール

## 概要

請求日と支払期日から売掛金の滞留日数を算出し、回収遅延先を抽出するツール。

## 起動

```powershell
cd 03_accounts_receivable_aging
python gui.py
```

## 使い方

1. 起動すると `data/receivables.csv` のサンプルデータを自動読み込みします
2. 「CSVを開く」で別のCSVに切り替えられます
3. 左側の一覧で明細を確認し、右側のグラフで傾向を把握します
4. 「結果を保存」で `results/receivable_aging_report.csv` に出力します

## サンプルデータ列

| 列名 | 内容 |
|------|------|
| 得意先 | 請求先企業 |
| 請求日 | 請求書発行日 |
| 支払期日 | 支払期限 |
| 請求額 | 請求金額 |
| 回収額 | すでに回収済みの金額 |

## 特徴

- 30日超・60日超・90日超の区分を自動で振り分けられる
- 得意先別の未回収残高を把握しやすい
- 督促優先順位づけのたたき台になる
